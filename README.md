# InternetGPS

基于 Taro 框架开发的跨平台应用，主要功能为实现联网 GPS 的实时查看

## 技术栈

- Taro
- React
- TypeScript
- Less

## 开始使用

### 环境要求

- Node.js (>= 12.0.0)
- npm 或 yarn

### 安装依赖

```bash
# 使用 npm
npm install

# 或使用 yarn
yarn
```

### 开发

```bash
# 开发微信小程序
npm run dev:weapp
```

### 打包

```bash
# 打包微信小程序
npm run build:weapp
```
